# 기여 가이드

## 🎯 가장 쉬운 기여: 미매핑 차명 정리

`data/unmapped.json`에는 인증은 존재하지만 소비자 모델명으로 연결되지 않은 차명이 있습니다.
아는 기종을 발견하면 `mapping/models.json`에 항목을 추가해 주세요.

```json
{ "nameKo": "혼다 골드윙", "brand": "혼다", "model": "골드윙", "aliases": ["GL1800"] }
```

- `nameKo`는 반드시 `"{brand} {model}"` 형식입니다 (CI가 검증합니다)
- `aliases`에 인증 차명(형식코드)을 넣으면 빌드가 인증 이력을 연결합니다
- PR 전에 `npm run build && npm run validate`를 실행하고 `data/` 산출물을 함께 커밋해 주세요

## ✍️ 표기 규칙

- 국내 공식 수입원 보도자료와 커뮤니티에서 통용되는 한글 표기를 따릅니다 (예: 뷸렛, 파니갈레)
- 배기량·연식 변형은 시장에서 별개 모델로 통하는 경우에만 분리합니다
